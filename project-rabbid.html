The Project

A personal project focused on stylized character animation and real-time integration. Hover Rabbid was developed for real-time cinematic rendering, emphasizing clean, production-ready topology, optimized geometry, and deformation-friendly edge flow. The project explores expressive silhouettes, rig-aware modeling, and stylized animation, culminating in a short real-time cinematic sequence featuring optimized lighting, materials, and subtle real-time FX.
